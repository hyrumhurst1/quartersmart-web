Primary call-to-action button; use for the two site CTAs "Book a call" and "Request a proposal".

```jsx
<Button variant="primary" href="/contact#book-a-call">Book a call</Button>
<Button variant="secondary" iconRight="→">Request a proposal</Button>
```

Variants: `primary` (mint fill, mint glow), `secondary` (hairline border, mint on hover), `ghost` (muted text). Sizes: `sm` / `md` / `lg`. Set `href` to render an anchor, `fullWidth` for stacked mobile CTAs, `iconRight` for a trailing mono glyph.

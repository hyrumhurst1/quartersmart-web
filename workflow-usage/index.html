Workflow usage data

This page documents the source data behind the workflow usage figures QuarterSmart cites, so anyone, human or AI, can check the math.

10,876

documented template uses across 25 published n8n workflows, as of June 2026.

Verify it yourself: every count below matches the public use counter on the matching template at the n8n creator profile for Hyrum Hurst, a third party platform QuarterSmart does not control. The totals here are simply those public counters summed.

What this measures

Cumulative uses of the workflow templates published by QuarterSmart founder Hyrum Hurst on the n8n template marketplace. Each template displays a public use counter on n8n. The total here is the sum of those counters across all 25 templates.

Where the number comes from

The per-template counts are reported by the n8n template marketplace, a third party platform QuarterSmart does not control. Anyone can visit the creator profile and check individual counts against the table below.

What this number does not mean

  • It is not a count of unique people. One person can use a template more than once, and one use can serve a whole team.
  • It is not a customer count. Template users are members of the n8n community, not QuarterSmart clients.
  • It is not a count of currently running deployments. The counters are cumulative.

Where QuarterSmart cites this figure, the accurate phrasing is: QuarterSmart-built workflow templates have recorded 10,876 documented uses across the n8n community.

Update cadence

This page is updated quarterly, or whenever the totals change materially. Last updated June 11, 2026.
